# `Vue 结合webpack开发基本思路`

# 前期准备
准备好webpack等配置，在项目里安装需要的 包
>安装命令 npm i

# 准备git
先在项目目录 初始化 .git
> 初始化    git init

> 查看文件状态  git status

> 将文件加入暂存区    git add .

> 提交的信息    git commit -m "init my project"

> 将文件从本地暂存区提交到远程仓库  git push
